Fix content not loading in editor: remove quill.update() that was overwriting content

Root cause: quill.update() re-renders from Quill's internal Delta model,
which was still empty after setting innerHTML. This overwrote our content
with the empty Delta, making the editor appear blank.

Fix:
- Use dangerouslyPasteHTML(html) as primary method (replaces all content)
- Set innerHTML directly as fallback (without calling update())
- Add retry after 200ms if editor still appears empty
- Add debug logging to trace loading issues
- Remove duplicate closing HTML tags
