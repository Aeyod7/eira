Fix image uploads on Vercel: switch from multipart to base64 JSON

Vercel serverless functions don't expose raw multipart body as a Buffer,
causing binary image data to be corrupted. Switch to base64-encoded JSON
which Vercel's body parser handles correctly. Also pin Node to 20.x to
silence engine warnings.
